Sitio web desarrollado con HTML5, CSS3 y JavaScript vanilla, sin frameworks ni librerías.
Técnicas: El sitio es totalmente responsive utilizando la técnica Mobile First.
Se utilizó JavaScript para guardar los datos ingresados por los eventuales aspirantes a asistir al evento en LocalStorage y 
para la validación correcta del formulario.
Para el deploy lo único que hay que realizar es simplemente ejecutar el archivo index.html.
